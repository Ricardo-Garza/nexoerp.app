# Paquete Soleil Wire para Nexo ERP

Contenido:
- `/imagenes_referencia`: referencias visuales nombradas por categoría.
- `manifest_imagenes.json` y `manifest_imagenes.csv`: mapa de identificadores.
- `SoleilWire_ERP_Import_Template.xlsx`: plantilla de carga masiva para catálogo, precios, CRM, inventario y oportunidades.

Uso sugerido:
1. Importar primero `Catalogo_Cables`.
2. Completar precios en `Lista_Precios`.
3. Cargar clientes/prospectos en `Clientes_CRM`.
4. Cargar oportunidades en `Oportunidades_CRM`.
5. Cargar inventario inicial si se desea controlar existencias.
6. Usar `Imagenes_Referencia` para vincular categoría/producto con imagen_id.

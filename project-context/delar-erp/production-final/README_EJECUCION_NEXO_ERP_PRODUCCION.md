# Ejecución del prompt definitivo

Copia los archivos en:

`project-context/delar-erp/production-final/`

Después, desde la raíz del repositorio, abre Claude Code y pega:

```text
Lee y ejecuta íntegramente:

project-context/delar-erp/production-final/PROMPT_DEFINITIVO_NEXO_ERP_PRODUCCION_PLUGINS.md

Usa todos los Plugins, Skills, MCP y conectores disponibles, verificando primero que funcionen.

Continúa hasta completar todas las puertas de calidad, fusionar a main y desplegar producción. Esta instrucción constituye autorización para merge y deploy cuando todo esté verde.
```

Modelo recomendado: el modelo de programación más potente disponible, con razonamiento alto o extendido y modo agente.
